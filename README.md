# 📄 Formulário de Pesquisa

Projeto desenvolvido com foco na criação de um formulário moderno, acessível e bem estruturado, aplicando boas práticas de **HTML5**, **CSS3**, **SEO** e **Acessibilidade**.

---

## 🚀 Sobre o projeto

Este projeto consiste na construção de um formulário de pesquisa interativo, com o objetivo de simular um cenário real de coleta de dados de usuários.

A aplicação foi desenvolvida priorizando:

- Estrutura semântica do HTML  
- Boas práticas de acessibilidade (A11y)  
- Técnicas básicas de SEO  
- Interface limpa e responsiva  

Além disso, o projeto reforça conceitos fundamentais para qualquer desenvolvedor front-end iniciante.

---

## 🛠️ Tecnologias utilizadas

- **HTML5** → Estrutura da aplicação  
- **CSS3** → Estilização e responsividade  

---

## 🎯 Conceitos aplicados

- Uso correto de `label` e `input` para acessibilidade  
- Organização semântica com tags HTML  
- Hierarquia de títulos (`h1`, `h2`, etc.)  
- Estrutura otimizada para SEO  
- Estilização com foco em experiência do usuário (UX)  
- Responsividade básica
